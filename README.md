# Gireevan Realty Website

A premium, nature-inspired real estate website for Gireevan Realty, showcasing NA plots and luxury bungalows in the Konkan region (Revoli-Pen, Raigad).

## Project Overview

This website is designed to provide potential customers with an immersive experience of the properties offered by Gireevan Realty. It emphasizes the connection with nature, featuring high-quality daylight imagery, a clean and trustworthy design, and clear calls to action.

**Live Preview:** Open `index.html` in your browser.

## Features

-   **Responsive Design:** Fully responsive layout that works seamlessly on desktops, tablets, and mobile devices.
-   **Premium UI/UX:**
    -   Nature-inspired color palette (Greens, Earthy tones).
    -   Smooth animations and transitions.
    -   Sticky header and mobile-friendly navigation.
-   **Showcase Galleries:** Masonry layout galleries for bungalows and project updates with lightbox integration.
-   **Lead Generation:** Integrated contact forms (powered by Formspree) and direct WhatsApp/Call links.
-   **Location Integration:** interactive Google Maps integration for easy navigation.
-   **Trust Signals:** Badges for "NA Approved", "clear title", and benefit highlights.

## Pages Included

1.  **Home (`index.html`):** Overview of offerings, amenities, and project highlights.
2.  **NA Plots (`na-plots.html`):** Detailed information about Non-Agricultural plots.
3.  **Bungalows (`bungalows.html`):** Showcase of bungalow typologies (1/2/3 BHK) and interiors.
4.  **Master Plan (`master-plan.html`):** "Coming Soon" page with project timeline and interest form.
5.  **Location (`location.html`):** Maps, directions, and connectivity details.
6.  **Gallery (`gallery.html`):** Visual tour of the project and surroundings.
7.  **Contact (`contact.html`):** Enquiry form and office details.
8.  **About (`about.html`)**: Company vision and mission.
9.  **FAQs (`faqs.html`)**: Common questions answered.

## Technology Stack

-   **HTML5:** Semantic markup.
-   **CSS3:** Custom styles with Flexbox/Grid (Variables used for theming).
-   **JavaScript (Vanilla):** Lightbox logic, sticky header, and form handling.
-   **Libraries:**
    -   [Font Awesome 6](https://fontawesome.com/) (Icons)
    -   [Google Fonts](https://fonts.google.com/) (Lato & Playfair Display)

## Setup & Usage

No build process is required. This is a static site.

1.  **Clone/Download** the repository.
2.  **Open** the `index.html` file in any modern web browser.
    -   *Recommended:* Use a local development server (e.g., Live Server in VS Code) for the best experience.
